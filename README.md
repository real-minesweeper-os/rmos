# Real Minesweepr Operating System

![Logo](./docs/logo.png)

Real Minesweepr Operating system project. 정말 말 그대로 지뢰찾기 운영체제 입니다. Real world의 지뢰를 찾는 것은 아니니 __군사적 목적은 전혀 없으며 단순 재미 및 농담으로 시작된 프로젝트 입니다.__
